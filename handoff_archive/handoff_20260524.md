## 技術的決定

| 決定項目 | 判断 | 理由 |
|---------|------|------|
| `git rm --cached` vs `git rm` | `--cached` を選択 | ローカルファイルが不要になるわけではない（開発に使う可能性あり） |
| git 履歴から完全削除 | 対象外 | BFG Repo Cleaner や `git filter-branch` が必要だが、現運用上のリスクは低い |
| `HELP.md` の処理 | 保持 | UI パネルの help テキストとして実装に使用されている可能性がある |
| Cloudflare Worker の扱い | 全削除 | Adobe Express アドオン本体とは独立したサービス（分離推奨） |
