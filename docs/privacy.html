プライバシーポリシー

最終更新: 2026年3月

はじめに

「Addon Explorer」（以下「本アドオン」）は、Adobe Express 向けのアドオン発見ポータルです。 本ポリシーは、本アドオンがどのような情報を収集・利用するかを説明します。

収集する情報

本アドオンは以下の情報のみを収集します。

  • 検索キーワードの集計カウント（例：「縦書き」が何回検索されたか）

収集は入力停止から 1.5 秒後にのみ行われ、2 文字以上のキーワードが対象です。

収集しない情報

  • IPアドレス
  • ユーザーID・セッションID
  • タイムスタンプ（いつ検索したか）
  • Adobe Express のアカウント情報
  • 個人を特定できる一切の情報

情報の利用目的

収集した集計データは、どのカテゴリのアドオンが求められているかを把握し、 将来のアドオン開発の参考にするためにのみ使用します。 第三者への販売・共有は行いません。

データの保管

集計カウントは Cloudflare Workers KV に保管されます。 生のキーワードログ（誰がいつ何を検索したか）は一切記録されません。

オプトアウト

本アドオンにはオプトアウト設定はありませんが、収集される情報は個人を特定しない 集計カウントのみであるため、プライバシーへの影響はありません。

お問い合わせ

本ポリシーに関するご質問は、Adobe Developer Console のプロフィールページに記載の 連絡先までお問い合わせください。

Privacy Policy

Last updated: March 2026

Introduction

"Addon Explorer" is an add-on discovery portal for Adobe Express. This policy explains what information the add-on collects and how it is used.

Information We Collect

The add-on collects only the following:

  • Aggregated search keyword counts (e.g., how many times "vertical text" was searched)

Collection occurs 1.5 seconds after the user stops typing, and only for keywords of 2 or more characters.

Information We Do NOT Collect

  • IP addresses
  • User IDs or session IDs
  • Timestamps (when a search occurred)
  • Adobe Express account information
  • Any personally identifiable information

How We Use the Information

Aggregated data is used solely to understand which types of add-ons are in demand, informing future add-on development. We do not sell or share this data with third parties.

Data Storage

Aggregated counts are stored in Cloudflare Workers KV. No raw keyword logs (who searched what and when) are ever recorded.

Opt-Out

There is no opt-out setting because the only data collected is non-identifiable aggregate counts, which carry no privacy risk.

Contact

For questions about this policy, please use the contact information listed on our Adobe Developer Console profile page.
