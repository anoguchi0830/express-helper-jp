# スクリーンショット撮影ガイド

Developer Console への申請に必要な 5 枚のスクリーンショットを撮影してここに配置する。

## 仕様

- **サイズ**: 1360 × 800 px
- **形式**: PNG または JPG（各 5 MB 以内）
- **言語**: 日本語 UI（locale = ja）

## 撮影リスト

| ファイル名 | 内容 | ポイント |
|-----------|------|----------|
| `01_home.png` | ホーム画面（グリッド表示） | 注目アドオンのカードが見える状態 |
| `02_search.png` | 検索結果画面 | 「デザイン」など日本語で検索した結果 |
| `03_category.png` | カテゴリ絞り込み画面 | カテゴリ一覧からクリック後のリスト |
| `04_modal.png` | アドオン詳細モーダル | モーダルが開いた状態。説明・キーワードが見える |
| `05_lang_switch.png` | 言語切替（EN / JA / KO） | LanguageSwitcher が見える状態。3言語表示 |

## 撮影手順

1. `npm run start` で開発サーバーを起動
2. Adobe Express に接続してパネルを開く
3. 言語を JA に切り替える
4. 各シーンを macOS スクリーンショット（Shift+Cmd+4）で撮影
5. 1360×800 にリサイズ（Preview.app → ツール → サイズを調整）
6. このディレクトリに配置
